# {{title}} — Decision Record

Append-only ADR log of load-bearing decisions and the nuances behind them, so
they **can't get lost**. **Chronological — oldest first (`D1` at top); append
each new `Dn` at the bottom.** One `Dn` per load-bearing call; every `Dn` also
marks its [[open-questions]] item resolved and drops a `decision` line in
[[log]]. See [[origination-workflow]].

## Index

_One line per decision, ascending — the latest is the highest number, at the
bottom of the file. Add a row when you add a `Dn`._
